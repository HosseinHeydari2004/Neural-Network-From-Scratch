# 🧠 Neural Network From Scratch

> Building a Neural Network from the ground up using only NumPy — no TensorFlow, no PyTorch, just mathematics and code.

---

## 🚀 About The Project

Most Deep Learning practitioners use frameworks like PyTorch and TensorFlow every day.

But what actually happens behind:

```python
model.fit(X, y)
```

This project answers that question by implementing the core building blocks of Neural Networks completely from scratch.

Every neuron, every weight update, every loss calculation, and every gradient is built manually using NumPy.

The goal is not to replace modern frameworks.

The goal is to understand them.
